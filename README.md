# Movie Selector

A simple app to list movies by genres. 
Visit [Pokedex](https://pokedex-beta-six.vercel.app/#/) deployed at Vercel.

## Run the app

`git clone git@github.com:Balazs-D/MovieSelector.git`

`cd Movie_Selector`

`npm install`

`npm start`

## Technologies

- React.js
- TypeScript
- Styled Components
- React Router
- Redux Toolkit
- Axios
- Babel
- Webpack
